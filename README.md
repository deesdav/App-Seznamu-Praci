# App-Seznamu-Praci

## Přehled

**App-Seznamu-Praci** je projekt zaměřený na vytvoření přívětivé aplikace pro správu ročníkových a maturitních prací.

## Funkce

- Správa prací: Vytváření, úprava a mazání prací.
